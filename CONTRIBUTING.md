# Contributing to Biometric Face Recognition Engine

Thank you for your interest in improving this project! We welcome contributions in the form of bug reports, feature requests, and pull requests.

## How to Contribute

1. **Report Bugs**: Use the GitHub Issue tracker.
2. **Feature Requests**: Open an issue to discuss new ideas.
3. **Pull Requests**: 
   - Fork the repository.
   - Create a feature branch.
   - Ensure your code follows the existing style.
   - Submit a PR with a clear description of changes.

## Development Setup

- Install dependencies: `pip install -r requirements.txt`
- Run tests: `python src/face_engine.py`

Please ensure all new features are documented in the `docs/` folder.
